# UE5 C++ RPG项目

## 简介
这是一个UE5的学习项目，使用GamePlay Ability System构建的游戏。

## 目标能力系统
- 1.轻攻击和重攻击的组合系统；
- 2.定向滚动；
- 3.防御；
- 4.格挡；
- 5.目标锁定；
- 6.具有冷却时间的特殊武器技能；
- 7.AI Avoidance(人工智能躲避)；
- 8.怪物的游荡人工智能；
- 9.怪物的武器发射；
- 10.boss战斗；

## 已实现的功能：
- 1.轻攻击和重攻击的组合；
- 2.小怪的受击表现以及属性更新；
